# exam1 실행 순서

## 구성
- `exam1.ino` : 아두이노 스케치. 100ms 주기로 카운터 값을 시리얼로 전송.
- `serial_camera_stream.py` : 파이썬 스크립트. OpenCV 카메라 스트림(FPS 표시) + 아두이노 시리얼 수신을 스레드로 동시 실행.

## 1. 아두이노 업로드
1. 아두이노 IDE에서 `exam1.ino` 열기.
2. 보드/포트 선택 후 업로드.
3. 업로드 완료 후에는 시리얼 모니터를 반드시 닫는다 (포트를 파이썬이 점유해야 하므로 열려 있으면 연결 실패).

## 2. 파이썬 환경 준비
```
pip install opencv-python pyserial
```

## 3. 포트 번호 확인
- Windows 장치 관리자 > 포트(COM & LPT) 에서 아두이노가 연결된 COM 번호 확인 (예: COM3).

## 4. 파이썬 스크립트 실행
```
python serial_camera_stream.py --port COM3 --baud 115200 --camera 0
```
- `--port` : 위에서 확인한 COM 포트로 변경.
- `--camera` : 사용할 카메라 장치 인덱스 (기본 0, 카메라가 여러 개면 0/1/2 등으로 변경).

## 5. 동작 확인
- OpenCV 창이 뜨고 카메라 화면에 FPS가 표시된다.
- 화면 좌측 상단에 아두이노가 보낸 카운터 값(`Arduino: ...`)이 계속 갱신된다.
- 종료하려면 카메라 창을 클릭해 포커스를 준 뒤 `q` 키를 누른다.

## 문제 해결
- `카메라를 열 수 없습니다` : `--camera` 값을 다른 인덱스로 변경하거나 다른 프로그램이 카메라를 사용 중인지 확인.
- `[Serial Error] ...` 가 화면에 표시됨 : 포트 번호가 맞는지, 시리얼 모니터 등 다른 프로그램이 포트를 점유하고 있지 않은지 확인.
- 값이 갱신되지 않음 : 아두이노 리셋 직후 부팅 대기(2초) 중일 수 있으니 잠시 대기.
